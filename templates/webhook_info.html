Webhook Configuration

Configure GitHub webhooks to enable real-time repository monitoring

GitHub Webhook URL

Use this URL in your GitHub repository webhook settings

Setup Instructions

  1. Navigate to your GitHub repository
  2. Click on Settings tab
  3. Select Webhooks from the left sidebar
  4. Click Add webhook button

Payload URL {{ webhook_url }}/webhook/github
Content type application/json

Environment Variables

Make sure these environment variables are configured on your Render deployment:

Variable Description Required
TELEGRAM_BOT_TOKEN Your Telegram bot token from @BotFather Required
GITHUB_TOKEN GitHub personal access token for API access Optional
GITHUB_WEBHOOK_SECRET Secret for verifying webhook authenticity Optional
WEBHOOK_URL Your deployed application URL Required
SESSION_SECRET Secret key for Flask sessions Required
